# Emoji Combiner Bot

A Discord bot that combines multiple emojis into a single PNG using Twemoji.

## Commands

- /emoji 😭💔
- /donate

## Setup

1. Copy `.env.example` to `.env`
2. Fill in values
3. Run:

docker compose up -d --build

## Features

- Private slash responses
- Twemoji rendering
- Docker ready
- Bitcoin + PayPal donations
