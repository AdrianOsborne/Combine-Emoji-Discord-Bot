# Emoji Combiner Bot

A Discord bot that combines multiple emojis into a single PNG using Twemoji.

## Setup

1. Clone '.env' with:
```
cp .env.example .env
```
2. Set DISCORD_BOT_TOKEN in `.env`
3. Run:
```
docker compose up -d --build
```

## Usage
```
/emoji :emoji1: :emoji2:
```
**Example**
```
/emoji 💔😭
```
